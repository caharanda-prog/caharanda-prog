# Technical Portfolio

The proof-of-concept projects in this portfolio showcase the technology stack I am currently working with and exploring.

Some of these PoCs are inspired by real production experience accumulated throughout my professional career. In other cases, they represent technical explorations where I apply the same architectural principles to modern practices within the backend, cloud, and distributed systems ecosystem.

The objective of this portfolio is not to solve a specific business use case, but to demonstrate design criteria, architectural decisions, configuration strategies, and practices commonly applied in professional environments.

This portfolio should be viewed as a demonstration of transversal technical capabilities rather than a collection of isolated projects.

## Infrastructure & Engineering Labs

Below are some of the technical labs currently included in this portfolio:

- **[Terraform Microservices Infrastructure Lab](https://github.com/caharanda-prog/terraform-microservices-infra-lab)**  
  Infrastructure-as-Code experiment modeling a containerized microservices architecture with Docker, service-level databases, and messaging components (RabbitMQ, Redis). Includes reusable modules for frontends, backends, and databases, designed to be easily replicated across environments.
