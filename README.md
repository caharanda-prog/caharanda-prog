The proof-of-concept projects in this portfolio showcase the technology stack I am
currently working with and interested in.

Some of these PoCs are based on real production experience throughout my professional
career; in other cases, I leverage that same background to adapt existing solutions to
new practices and trends within the backend, cloud, and modern architecture ecosystems.
Additionally, this portfolio includes technical explorations of technologies that I
consider relevant in today’s industry.

The main goal of this portfolio is not to solve a specific business use case, but to
demonstrate design criteria, architectural decisions, configuration approaches, and
best practices commonly applied in professional environments.

This portfolio is intended as a **portfolio of my transversal technical capabilities**,
rather than a collection of isolated projects.
